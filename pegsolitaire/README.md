# PEG Solitaire

## Problem:
```
 English          European
    O O O             O O O
    O O O           O O O O O
O O O O O O O     O O O O O O O
O O O . O O O     O O O . O O O
O O O O O O O     O O O O O O O
    O O O           O O O O O
    O O O             O O O
```
You should remove all pegs by moving them. A valid move is to jump a peg orthogonally over an adjacent peg into a hole two positions away and then to remove the jumped peg. The game is won when only one peg is left.

## Program:
### IN:
 * L number of lines
 * L lines containing a valid board using characters (# = No board, O = peg, . = hole)
```
7
##OOO##
##OOO##
OOOOOOO
OOO.OOO
OOOOOOO
##OOO##
##OOO##
```
